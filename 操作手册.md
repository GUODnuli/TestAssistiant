# AI测试用例转换器快速使用指南

## 简单三步使用法

### 1. 配置环境
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env 文件，填入你的DeepSeek API密钥
```

### 2. 启动服务
```bash
# 启动服务
python main.py
```

### 3. 使用API
- 访问 http://localhost:8000/docs 查看API文档
- 使用 `/api/v1/convert-test-case` 转换单个测试用例
- 使用 `/api/v1/batch-convert` 批量转换测试用例
